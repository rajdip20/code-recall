This exercise is a two-parter. Double the fun (and double the skill-building)!

Exercise: Conditional Calculations

Part 1:

  1. Download and open the "Home Sale Data.xlsx" file, found in the resources for this section. Review the data on the "Home Sale Data" worksheet, which lists all the homes sold by a particular real estate agency in 2020.

  2. Using the Ctrl + Shift + Down Arrow keyboard shortcut, select all the agent names from the "Agent" column, then copy and paste these names into cell A2 of the "Exercise 1" worksheet.

  3. Using Excel's "Remove Duplicates" feature, transform this list of names into just the distinct/unique agent names (there should be 12).

  4. Using Excel's SUMIFS function, calculate the total 2020 sales by each real estate agent in the "Total 2020 Sales" column on the "Exercise 1" worksheet.

Part 2:

  1. Using the Ctrl + Shift + Down Arrow keyboard shortcut, select all the city names from the "City" column of the "Home Sale Data" worksheet, then copy and paste these cities into cell A2 of the "Exercise 2" worksheet.

  2. Using Excel's "Remove Duplicates" feature, transform this list of cities into just the distinct/unique agent names (there should be 11).

  3. In columns B through D of the "Exercise 2" worksheet, use Excel's COUNTIFS function to calculate the total sales for each city in 2020, per the sale price threshold specified in each column header. For example, in column B, you'll only want to count home sales for the given city in column A, if the sale price was less than $500,000. HINT: you will need to apply multiple criteria in your COUNTIFS functions to get this to work.

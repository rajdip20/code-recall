Formulas are where the magic starts to happen, where you can really start doing some wizardry with data. Use this exercise to practice your formulas.

Exercise: Excel Formulas 101

  1. Download and open the "Excel Formulas - Starter.xlsx" file directly below (at the bottom of this page).

  2. On the "Exercise 1" worksheet, insert a blank column to the left of the "Product Name" column, and give it the column header "Days Since Purchase".

  3. Populate this new column with a formula that calculates the elapsed days between the purchase date and the review date. Make sure the results are formatted as numbers, with 0 digits after the decimal.

  4. Add a new column to the right of "Customer Last Name", with the column header "Username".

  5. Populate this column with a formula that automatically generates customer usernames as follows: it should concatenate the customer's first name, then a period/dot, then the customer's last name, in a single string.

  6. On the "Exercise 2" worksheet, fill out the multiplication table by writing a single formula in the top left blank cell of the table, then copying that formula over the rest of the table. Each cell in the table should yield the result of multiplying the column header for that cell by its row header.

  7. You will need to employ some combination of absolute and relative cell references in your formula to accomplish this. You can preview what the solution would look like (with no formulas of course!) on the "Multiplication Table Example" worksheet.

Solution: download the "Excel Formulas - Exercise Solution.xlsx" file to check your work.
